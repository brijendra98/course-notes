# CS241 - LEC003: Machine Language/Assembly Language

Example: Add 42 to 52, store the result in $3.

Load Immediate and Skip (lis)
---> lis $d
---> 00000 00000 00000 ddddd 00000 010100
```
$d <--- MEM[PC]; -PC = PC+4


     lis $5          0x00002814
---> .word 42        0x0000002A
     lis $7          0x00003814
     word 52         0x00000034
     add $3, $5, $7  0x00A71820
     jv $31
```
